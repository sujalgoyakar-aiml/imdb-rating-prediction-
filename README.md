# IMDb Rating Prediction 

## About
Built a machine learning model to predict IMDb ratings 
of top 1000 movies using real historical data.

## Key Insights
Number of Votes is the strongest predictor of rating
 Animation Drama War genre has highest average ratings
 Frank Darabont has highest average director rating (8.95)
 Predicted rating for a GOT-like movie: 8.7/10

## Model Performance
 Algorithm: Random Forest Regressor
 R2 Score: 0.55

## How To Improve
 Add Genre and Director as encoded features
 Use larger dataset (10,000+ movies)
 Try XGBoost or Gradient Boosting
